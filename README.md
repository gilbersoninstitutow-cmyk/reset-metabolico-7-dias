# Reset Metabólico 7 Dias

Produto low ticket do ecossistema W Metabolic Performance 360º.

## Objetivo

Criar uma oferta simples, direta e acessível para pessoas que:
- se sentem cansadas
- inchadas
- inflamadas
- sem energia
- com baixa disposição

A proposta é entregar um plano simples e prático para iniciar uma mudança perceptível em 7 dias.

---

## Estrutura Inicial

- Landing page simples
- Criativos estáticos
- Carrossel
- Checkout
- Meta Ads
- Funil de entrada
- Captação via WhatsApp ou página de venda

---

## Prioridade Estratégica

Conversão antes de complexidade visual.

O foco inicial NÃO é:
- VSL complexa
- animações
- cinematic excessivo

O foco é:
- oferta
- copy
- conversão
- tráfego
- validação
